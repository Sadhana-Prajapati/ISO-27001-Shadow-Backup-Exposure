💀 Shadow Backup Exposure

ISO 27001-Based CTF Challenge

In a small organization, a misconfigured backup server was exposed on the public internet. Your task is to identify the vulnerability, extract the sensitive data, and understand how it relates to ISO 27001 controls.

🧩 Challenge Objectives

  • Perform recon on the target service.
  • Identify misconfigured services (FTP, HTTP).
  • Access sensitive backup files or credentials.
  • Find and submit the flag.

📜 ISO 27001 Mapping

This CTF challenge maps to the following ISO 27001:2022 controls:

  • A.9.2.6 – Secure log-on procedures
  • A.12.1.1 – Documented operating procedures
  • A.13.1.1 – Network controls
  • A.14.2.7 – Outsourced development and data handling

🚩 Flag Format

Submit your flag in this format: CTF{example_flag_here}

📥 How to Play

  1. Clone the repo or view challenge files on GitHub.
  2. Use tools like nmap, ftp, wget, and strings to explore.
  3. Find the exposed backup or credential file and extract the flag.
  4. Submit the flag as per format.
